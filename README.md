## API

### 堺市 企業オープンデータ取得API

- https://io.sakacil.com/api/info/list?{"offset":0,"limit":100}
- offset: オフセット
- limit: 取得件数

## ライセンス

- 本データは、CC BY、ODC-By、ODbL のいずれかに従ってご利用いただけるオープンデータです。
